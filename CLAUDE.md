# Awesome Hacking

## Overview
Curated collection of awesome hacking resources, tools, tutorials, and frameworks for security researchers, penetration testers, and ethical hackers.

## Repository Purpose
- **Security research** - Comprehensive hacking resource directory
- **Penetration testing** - Tools and techniques
- **Ethical hacking** - Legal and authorized security testing
- **Learning resource** - Security education and training

- **Subdomain Enumeration** - Sublist3r, Amass

- **Password Cracking** - John the Ripper, Hashcat, Hydra
- **Privilege Escalation** - Linux/Windows escalation tools

### Vulnerability Analysis

- **Web Scanners** - SQLMap, XSSer, Commix
- **Code Analysis** - Bandit, Semgrep, SonarQube
